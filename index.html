1. Introdução ao Python

Python é uma linguagem de programação de alto nível, criada por Guido van Rossum e lançada em 1991. Sua sintaxe clara e legível a torna ideal tanto para iniciantes quanto para profissionais. É amplamente utilizada em diversas áreas: desenvolvimento web, ciência de dados, automação, inteligência artificial, cibersegurança e muito mais.

Curiosidades e Vantagens do Python
  • Simplicidade: A sintaxe se aproxima do pseudocódigo, facilitando o aprendizado.
  • Versatilidade: Amplamente utilizada em diversos domínios, desde scripts simples até aplicações complexas.
  • Comunidade Ativa: Milhares de bibliotecas e uma vasta comunidade ajudam na resolução de problemas.

2. Instalação e Ambiente

A seguir, veja como instalar e configurar seu ambiente para desenvolver em Python.

2.1. Windows

  1. Acesse o site oficial: python.org/downloads.
  2. Baixe o instalador (versão 3.x) e execute-o.
  3. Marque a opção "Add Python to PATH" antes de instalar.
  4. Para verificar, abra o Prompt de Comando e digite: 
    python --version

2.2. Linux/macOS

No terminal, execute:

sudo apt install python3

Verifique a instalação com:

python3 --version

2.3. Escolhendo um Editor/IDE

Recomendamos utilizar editores como VS Code, PyCharm ou Jupyter Notebook para uma experiência de desenvolvimento mais rica.

3. Conceitos Básicos

Nesta seção, você aprenderá sobre variáveis, tipos de dados, entrada/saída e operadores.

3.1. Variáveis e Tipos de Dados

Em Python, não é necessário declarar o tipo da variável; ele é inferido automaticamente.

idade = 25          # Inteiro
altura = 1.75         # Float
nome = "Alice"        # String
ativo = True          # Booleano
Exemplo Interativo: Exibindo Variáveis

Experimente copiar e executar o seguinte código em seu interpretador Python:

nome = input("Digite seu nome: ")
print("Olá,", nome)

3.2. Entrada e Saída

Utilize input() para ler dados e print() para exibi-los.

numero = int(input("Digite um número: "))
print("Você digitou:", numero)

3.3. Operadores

  • Aritméticos: +, -, *, /, //, %, **
  • Comparação: ==, !=, >, <, >=, <=
  • Lógicos: and, or, not
a = 10
b = 3
print(a + b)      # Soma: 13
print(a ** b)     # Exponenciação: 1000

4. Estruturas de Controle

As estruturas de controle permitem direcionar o fluxo do programa.

4.1. Condicionais (if, elif, else)

idade = int(input("Digite sua idade: "))
if idade >= 18:
    print("Maior de idade")
else:
    print("Menor de idade")

4.2. Laços de Repetição

Os laços for e while são usados para repetir ações.

# Exemplo com for:
for i in range(5):
    print("Número", i)

# Exemplo com while:
contador = 0
while contador < 5:
    print("Contador:", contador)
    contador += 1
Quiz Interativo

Qual dos seguintes códigos imprime os números de 0 a 4?

Escolha a alternativa correta:

  1. A: for i in range(1,5): print(i)
  2. B: for i in range(5): print(i)
  3. C: while i < 5: print(i)

Resposta: Alternativa B

5. Funções

Funções são blocos de código reutilizáveis. Elas podem receber parâmetros e retornar valores.

def saudacao(nome):
    return "Olá, " + nome + "!"

print(saudacao("Alice"))

6. Estruturas de Dados

Python possui diversas estruturas de dados poderosas para armazenar e manipular informações.

6.1. Listas

frutas = ["maçã", "banana", "laranja"]
print(frutas[0])    # Imprime "maçã"
frutas.append("uva")
print(frutas)

6.2. Tuplas

cores = ("vermelho", "verde", "azul")
print(cores[1])    # Imprime "verde"

6.3. Dicionários

pessoa = {"nome": "Alice", "idade": 30}
print(pessoa["nome"])

6.4. Conjuntos

numeros = {1, 2, 3, 3, 2}
print(numeros)  # Conjuntos não repetem valores

7. Módulos e Pacotes

Para organizar seu código e reutilizá-lo, Python permite que você divida seu programa em módulos e pacotes.

# Exemplo de importação de módulo

8. Programação Orientada a Objetos

A POO permite organizar programas por meio de classes e objetos. Veja um exemplo básico:

class Pessoa:
    def __init__(self, nome, idade):
        self.nome = nome
        self.idade = idade

    def apresentar(self):
        return f"Olá, meu nome é {self.nome} e tenho {self.idade} anos."

p = Pessoa("Alice", 30)
print(p.apresentar())
Conceitos Importantes
  • Encapsulamento: Protege os dados dos objetos.
  • Herança: Permite criar classes derivadas com funcionalidades extras.
  • Polimorfismo: Permite que métodos com o mesmo nome se comportem de maneira diferente em classes distintas.

9. Tópicos Avançados

Nesta seção, abordamos conceitos mais avançados para quem deseja se aprofundar na linguagem.

9.1. Decorators

Decorators são funções que modificam o comportamento de outras funções sem alterá-las diretamente.

def meu_decorator(func):
    def wrapper(*args, **kwargs):
        print("Executando antes da função")
        resultado = func(*args, **kwargs)
        print("Executando depois da função")
        return resultado
    return wrapper

@meu_decorator
def dizer_oi():
    print("Oi!")

dizer_oi()

9.2. Geradores

Geradores permitem criar iteradores de forma simples e econômica em termos de memória.

def contador(max):
    n = 0
    while n < max:
        yield n
        n += 1

for numero in contador(5):
    print(numero)

9.3. Context Managers

Context managers (gerenciadores de contexto) são usados para gerenciar recursos, como arquivos, garantindo que sejam fechados corretamente.

with open("arquivo.txt", "r") as arquivo:
    conteudo = arquivo.read()
    print(conteudo)

Coloque em prática o que você aprendeu resolvendo os exercícios abaixo:

  1. Olá, Mundo! – Crie um programa que exiba "Olá, Mundo!" na tela.
    Mostrar Solução 
    print("Olá, Mundo!")
  2. Saudação Personalizada – Peça o nome do usuário e exiba uma saudação.
    Mostrar Solução

print("Olá,", nome)
  3. Soma de Dois Números – Leia dois números e exiba a soma.
    Mostrar Solução 
    num1 = int(input("Digite o primeiro número: "))
num2 = int(input("Digite o segundo número: "))

  4. Conversão de Temperatura – Converta uma temperatura de Celsius para Fahrenheit.
    Mostrar Solução 
    celsius = float(input("Temperatura em °C: "))
fahrenheit = celsius * 9/5 + 32
print(f"{celsius} °C = {fahrenheit} °F")
  5. Desafio Avançado: Crie uma função que, dado um número, retorne seu antecessor, número e sucessor em forma de tupla.
    Mostrar Solução 
    def sequencia(num):
    return (num - 1, num, num + 1)

numero = int(input("Digite um número: "))
print(sequencia(numero))
