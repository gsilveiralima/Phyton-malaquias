Introdução ao Python - Curso Completo

📌 O que é Python?

Python é uma linguagem de programação de alto nível, interpretada e orientada a objetos. Foi criada no final dos anos 1980 e é amplamente usada em diversas áreas, incluindo desenvolvimento web, ciência de dados, automação, inteligência artificial e cibersegurança.

📌 Instalação do Python

Windows

  1. Acesse o site oficial: Python.org.
  2. Baixe o instalador e execute-o.
  3. Marque a opção "Add Python to PATH" antes de instalar.
  4. Após a instalação, abra o prompt de comando e digite: 
    python --version

Linux/macOS

No terminal, digite:

sudo apt install python3

Para verificar a instalação:

python3 --version

📌 Primeiros Comandos

Após instalar Python, podemos rodar o interpretador interativo:

python

Agora, tente os seguintes comandos:

print("Olá, Mundo!")

📌 Variáveis e Tipos de Dados

Python permite criar variáveis sem declarar tipos. Exemplos:

idade = 25  # Inteiro
altura = 1.75  # Float
nome = "Alice"  # String
ativo = True  # Booleano

📌 Entrada e Saída de Dados

Podemos capturar entrada do usuário com input() e exibir com print():

nome = input("Digite seu nome: ")
print("Bem-vindo,", nome)

📌 Operadores Matemáticos

  • Adição: +
  • Subtração: -
  • Multiplicação: *
  • Divisão: /
  • Exponenciação: **
  • Módulo (resto): %
a = 10
b = 3
print(a + b)  # 13
print(a ** b)  # 1000

📌 Exercícios

Pratique com os exercícios abaixo:

  1. Crie um programa que exiba "Olá, Mundo!".
  2. Peça o nome do usuário e exiba uma saudação.
  3. Peça dois números e mostre a soma deles.
  4. Converta uma temperatura de Celsius para Fahrenheit.
